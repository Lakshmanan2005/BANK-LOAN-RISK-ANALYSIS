# BANK-LOAN-RISK-ANALYSIS

This project analyzes bank loan data to identify high-risk customers and loan segments using **Python, SQL, and Power BI**.  
The goal is to help banks reduce default risk by understanding patterns based on **Credit Score, LTV, Region, and Loan Type**.

---

## Project Objective
To build an interactive dashboard that:
- Monitors loan portfolio performance
- Identifies high-risk regions and customer segments
- Understands how Credit Score, LTV, and Loan Type affect defaults
- Supports data-driven decision making for loan approvals

---

## Tools & Technologies
- **Python** → Data cleaning, preprocessing, feature engineering  
- **SQL** → Data querying, aggregation, and business logic  
- **Power BI** → Dashboard creation and visualization
